ABOUT:
This program is part of the final group project of the Professional Qualification Course 
for Applied Programming undertaken in 2020.
Written in the C# programming language, using the ASP.NET Framework.

GENERAL INFO:
This web application enables you to create an account and log in order to reserve flights.
The admin can add flights and also promote users to enable them to add flights as well.

STRUCTURE:
This project uses a local database on an SQL Server.
It is made using the MVC architecture separating the Model, Views and Controllers.

The project was created in an agile environment following the different steps in the scrum methodology.
